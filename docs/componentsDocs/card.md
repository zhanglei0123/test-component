# Card
卡片组件

### 示例
<m-card imgSrc="/testone.jpeg" summary="test" />

### 代码
```html
<m-card imgSrc="/testone.jpeg" summary="test" />

```

### Attributes
| 参数 | 说明 | 类型 | 是否必要 | 默认值 |
| --- | --- | --- | --- | --- | --- |
| width | 宽度 | Number | false | - |
| imgSrc | 图片资源 | String | true | - |
| imgHeight | 图片高度 | Number | false | - |
| summary | 概要 | String/slot | false | - |
| footer | 底部 | slot | false | - |
